# Comingsoon collabus

Comming soon page for Collabus 

## Requirements

- debian
- docker
- docker compose

## Deployment

### Create **.env** file in project root folder 

```bash
# example of .env file
FRONTEND_PORT=80
BACKEND_PORT=3000
BACKEND_URL=http://localhost:3000/
SECURE_KEY=QW32Jk369wsFP25kbSdt
```

### Commands

```bash
# from project root folder
./start.sh
./stop.sh
```
